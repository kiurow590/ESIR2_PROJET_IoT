## Généralités

Ce projet est dans le cadre de l'UE FabLab.

Ce repo se subdivise en deux parties :

- La partie [IHM](https://github.com/ESIR-IOT-2-Project/project-teamzigbee/tree/main/IHM), qui est un site web permettant de gérer les capteurs et les actionneurs. (FrontEnd : VueJS) 
- La partie [Serveur](https://github.com/ESIR-IOT-2-Project/project-teamzigbee/tree/main/Server), qui est un serveur permettant la gestion et la sauvegarde des données. (BackEnd : NodeJS)
