La partie IHM est constitué de plusieurs dossiers, les fichier qui servent à la gestion 
du site se trouvent dans le dossier src.

Pour lancer le site, il faut lancer la commande suivante dans le dossier IHM :
```bash
sudo npm install
npm run dev
```

Vous accederez au site en tapant l'adresse suivante dans votre navigateur :
```bash
http://localhost:5173/
```
